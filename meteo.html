🌦️ Weather App

Une application mobile React Native pour consulter la météo en temps réel

Présentation du projet

Weather App est une application développée en React Native permettant à l’utilisateur de consulter la météo actuelle d’une ville grâce à une API météo (par exemple OpenWeatherMap). L’interface simple et épurée permet de visualiser les données météo en un clin d’œil.

Fonctionnalités principales

  • 📍 Recherche météo par ville
  • 🌡️ Température, humidité, vitesse du vent
  • 🔄 Mise à jour en temps réel via API
